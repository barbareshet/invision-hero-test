what is schema.org?

Welcome to Schema.org

Schema.org is sponsored by Google, Microsoft, Yahoo and Yandex.
Schema.org vocabulary can be used with many different encodings, including RDFa, Microdata and JSON-LD. These vocabularies cover entities, relationships between entities and actions, and can easily be extended through a well-documented extension model

The Organization of Schemas - Inheritance

he schemas are a set of 'types', each associated with a set of properties. The types are arranged in a hierarchy
It all start from "THING" and everything bellow inherits form the "THING".
For example, if one want to do a Business Event and mark it with microdata, the BusinessEvent inherits all properties of the base Event (the parent), which also inherits the properties of THING". Schema Full Hierarchy
In a nutshell: every schema inherits the properties of its parents, and adds extra properties on top of them
